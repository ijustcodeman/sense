# Quellenverzeichnis

Folgende Quellen und Dokumentationen wurden für die Bearbeitung der Aufgabe verwendet:

1.  **Android Developers - Sensors Overview**
    *   Thema: Grundlagen der Sensor-Programmierung in Android.
    *   URL: [https://developer.android.com/guide/topics/sensors/sensors_overview](https://developer.android.com/guide/topics/sensors/sensors_overview)

2.  **Android Developers - Motion Sensors**
    *   Thema: Spezifische Nutzung des Beschleunigungssensors (`TYPE_LINEAR_ACCELERATION`).
    *   URL: [https://developer.android.com/guide/topics/sensors/sensors_motion](https://developer.android.com/guide/topics/sensors/sensors_motion)

3.  **Android Developers - Notifications Overview**
    *   Thema: Erstellung von Notifications und Notification Channels ab API Level 26.
    *   URL: [https://developer.android.com/develop/ui/views/notifications](https://developer.android.com/develop/ui/views/notifications)

4.  **Android Developers - Handler & Looper**
    *   Thema: Zeitgesteuerte Aufgabenwiederholung für den "Gravity"-Effekt.
    *   URL: [https://developer.android.com/reference/android/os/Handler](https://developer.android.com/reference/android/os/Handler)

5.  **Android Developers - ProgressBar**
    *   Thema: Nutzung und Konfiguration des horizontalen Fortschrittsbalkens.
    *   URL: [https://developer.android.com/reference/android/widget/ProgressBar](https://developer.android.com/reference/android/widget/ProgressBar)
